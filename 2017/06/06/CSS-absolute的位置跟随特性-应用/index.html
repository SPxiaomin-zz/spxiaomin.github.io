跟随性讲解

如果元素本身就是 block 和其他元素不在一行，那么 absolute 之后也不会在同一行。如果元素本身是 inline or inline-block 和其他元素在一行，那么 absolute 之后也是在同一行的。简单就是说原来在什么位置，绝对定位之后还是在什么位置。
