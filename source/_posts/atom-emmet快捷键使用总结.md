- ! + `tab`

    这样就可以自动生成一个 `html5` 的文档结构。

- meta:vp + `tab`

    ```html
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    ```

- meta:compat + `tab`

    ```html
    <meta http-equiv="X-UA-Compatible" content="IE=7">
    ```

- link + `tab`

    ```html
    <link rel="stylesheet" href="">
    ```

- script + `tab`

    ```html
    <script></script>
    ```

- script:src + `tab`

    ```html
    <script src=""></script>
    ```

- 选中内容　+ `ctrl+alt+w`

    就可以用新的元素包裹内容.
